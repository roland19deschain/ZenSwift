## Requirements:
- **iOS** 9.0+ / **tvOS** 9.0+ / **watchOS** 2.0+ / **macOS** 10.10+
- Xcode 9.0+
- Swift 4.0+

## Installation

ZenSwift is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZenSwift'
```
and run `pod install`
